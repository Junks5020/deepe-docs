# 导航页 - 系统总览与技术指标

## 1. 页面概述

导航页作为系统的核心数据总览界面，为用户提供全面的训练与评估活动监控。本页面通过多维度的数据可视化组件，实时展示关键性能指标（KPIs）和系统使用模式分析。

## 2. 核心指标仪表盘

### 2.1 基础统计指标
- **微调训练总次数**：累计执行的模型微调任务数量
- **模型评估次数**：完成的模型性能评估与测试任务总量
- **内部数据集数量**：系统管理的专有数据集资源总数
- **知识库数量**：构建的知识图谱与文档库实例
- **基础模型数量**：可用的基础预训练模型总数
- **定制模型数量**：经微调处理的个性化模型数量
- **完整模型数量**：完成全流程训练的成熟模型数量
- **上线模型数量**：已部署至生产环境的模型数量

## 3. 数据可视化组件

### 3.1 训练活动趋势图

**组件类型**：时间序列柱状图

**数据维度**：过去7天

**指标说明**：

- X轴：日期时间戳（精确到天）

- Y轴：当日微调训练执行次数

- 数据点：每日训练任务计数

### 3.2 基础模型使用分布

**组件类型**：环形饼图

**统计维度**：各基础模型在微调任务中的被选用频率

**数据来源**：微调任务配置日志

### 3.3 评估模式分布图

**组件类型**：比例标签矩形树图

**视觉形式**：多个彩色矩形方块平铺排列

**数据编码**：

- 矩形面积：代表该评估模式的使用频率或比例

- 矩形颜色：区分不同的评估模式类型（如：裁判模式、评估模式、单模型模式等）

- 矩形内文字：标注评估类型名称及其具体百分比数值

**业务价值**：直观展示各评估模式的占比，快速识别当前使用频率较高的主流评估策略。

### 3.4 第三方模型供应商分析

**组件类型**：多层饼图（Sunburst Chart）

**层级结构**：

- 内层：模型供应商分布（如：ollama、硅基流动、黑森林实验室等）

- 外层：各供应商下的具体模型使用情况
**色彩编码**：不同供应商使用对比色系，同供应商下模型使用渐变色
