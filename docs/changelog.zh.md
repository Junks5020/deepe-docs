## 🆕 Version 2511 (最新版本)

**发布日期：** 2025年11月

### ✨ 用户体验优化

#### 🔄 训练流程改进

- **实时状态同步**：训练、保存、部署等任务提交处理完成后立即显示在模型列表，无需先去检查任务状态

- **快速错误处理**：训练失败时立即显示错误信息并提供中断训练选项

#### 📊 模型评估增强

- **智能错误处理**：友好的错误提示和具体的修改建议

- **部分成功支持**：即使部分样本失败，仍可查看其他样本结果


#### 🎯 新增微调案例

- **Qwen大语言模型案例**：人物生平训练实战

- **Stable Diffusion案例**：古早头像风格图像生成实战

#### 🐛 问题修复

- 修复模型评估预览功能显示异常

- 优化训练状态检测机制

- 改进界面加载性能

## 🚀 Version 2509

**发布日期：** 2025年9月

### 核心升级

#### 🔄 智能生命周期管理

- **非线性工作流**：支持自由设计的模型迭代路径

- **完整版本控制**：自动记录训练参数、数据和结果

- **模型谱系追踪**：清晰查看模型演进历程

#### 🛠️ 环境与部署

- **自定义训练环境**：灵活配置依赖库和计算资源

- **无缝部署**：支持Ollama等标准平台

- **容器化支持**：预配置Stable Diffusion 3.5 Medium环境

#### 🎨 模型类型扩展

- **图像生成模型**：基于Stable Diffusion技术，支持文生图、图生图

- **多模态模型**：统一平台下的全生命周期管理

#### 💼 企业级特性
- 多角色团队协作

- 企业合规支持

## 👁️ Version 2507

**发布日期：** 2025年7月

### 重大更新

#### 📸 多模态视觉模型

- **全链路支持**：Qwen-VL、Llama-Vision等视觉模型的微调、推理、部署

- **数据集可视化**：多图配置，原生可视化Bounding Box

- **视觉监督式微调**：自定义数据启动微调任务

#### 🎯 DeepPrompt增强

- 全面支持图生文场景

- 单图/多图输入支持

- 自定义任务指令

#### ⚙️ 训练灵活性

- **参数完全可定制**：支持string、int、float、bool类型

- **前端直接配置**：参数直接传入Python脚本

#### 🔍 评估能力升级

- 多图推理支持

- 自动对比预测结果

- 图文内容同时展示
